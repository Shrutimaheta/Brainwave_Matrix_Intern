# ATM Interface Application

This is a simple Java Swing-based ATM interface application that simulates basic banking operations with a graphical user interface.

## Features

- **Login Screen:** Secure login to access the ATM.
- **Main Menu:** Options to Deposit, Withdraw, Check Balance, View Transaction History, and Exit.
- **Deposit:** Add money to the account.
- **Withdraw:** Withdraw money from the account.
- **Balance:** Display current account balance.
- **Transaction History:** View past transactions.

## Technologies Used

- Java
- Swing (for GUI)
